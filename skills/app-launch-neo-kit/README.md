# app-launch-neo-kit-public

A sanitized public version of the app launch skill.

Before publishing, replace placeholders in `SKILL.md`:

- `{SITE_NAME}`
- `{SITE_BASE_URL}`
- `{SITE_DOMAIN}`
- `{OUTPUT_DIR}`
- `{DEPLOY_KIT_DIR}`
- `com.example.<slug>`

Do not commit real FTP, WordPress, analytics, or form credentials.
